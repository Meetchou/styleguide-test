# Introduction

This project uses Tailwindcss so a lot of colour props are passed in as CSS classes that
Tailwind applies styling to i.e. `text-primary-500` to use the primary blue. You can find the
full original colour palette [here](https://tailwindcss.com/docs/customizing-colors/#default-color-palette).
This palette is generate by [tailwind.config.js](https://github.com/tortugeniale/styleguide/blob/master/tailwind.config.js)
